EnterFixer

EnterFixer is a web-based tool designed to streamline text formatting. It allows users to convert unwanted paragraph breaks into "Shift + Enter" style line breaks with a single click. Additionally, it offers case conversion features, enabling easy changes between UPPERCASE, lowercase, Sentence case, and more. This tool enhances productivity for writers, editors, and anyone dealing with text formatting.

WordLok

WordLok is a 2D word puzzle game that challenges players to find all the letters in a 5-letter word before exhausting 11 moves. Players are presented with a grid where the letters of the word are randomly placed among other letters. After a brief memorization period, the letters are hidden, and players must recall their positions to succeed. This game aims to improve vocabulary and memory in a relaxing, easy-to-play environment.
